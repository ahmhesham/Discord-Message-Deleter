Deletes messages in a chat using hotkeys.

## Hotkeys

- **Ctrl + D** — start deleting  
- **Esc** or **Ctrl + S** — stop deleting

## Requirements

- AutoHotkey v2.0 installed
  https://www.autohotkey.com/

## Usage

1. Run the `.ahk` script
2. Focus the chat window
3. Press **Ctrl + D** to start
4. Press **Esc** or **Ctrl + S** to stop

## Notes

- Works only when the chat is focused
- Behavior depends on the chat app
- Deleted messages is permanent!@!
